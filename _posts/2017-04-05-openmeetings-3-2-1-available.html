Version 3.2.1 of OpenMeetings changes the UI to use a mixed mode HTML5 and is part of the roadmap to a full HTML5 Web-Conferencing client.



The new features in HTML5 include:

  • Room menu
  • User list
  • File management
  • Activities and actions
  • Room chat



The remaining part using the Flash plugin is the whiteboard and Audio/Video.



Additionally v3.2.1 improves some of the video streaming, upload and handling of documents in the whiteboard and enhancements around the chat and calendar.



Download and documentation available at: openmeetings.apache.org/downloads.html
