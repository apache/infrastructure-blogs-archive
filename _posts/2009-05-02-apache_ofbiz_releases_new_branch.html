One of the top enterprise software packages in the world,
OFBiz boasts hundreds of experienced service providers and thousands of live
production instances; the framework comprises mature eCommerce, Enterprise
Resource Planning, and Customer Relationship Management functionality with a
general enterprise software framework and a set of base and special purpose
applications. Release Branch 9.4 features numerous enterprise-grade
enhancements, as well as a new time-based version system based on the date of
release, similar to the Ubuntu Linux version numbers. The latest version is
known as "Release9.04" because it has branched from the main code
base (the "trunk") in April 2009.
