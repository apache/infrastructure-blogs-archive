Another productive week at The Apache Software Foundation! Here's what happened within the Apache Community:

ASF Board –the ASF Board is responsible for management and oversight of the business and affairs of the corporation in accordance with the Foundation's Bylaws.
 - Next Board Meeting: 18 November. Board calendar and minutes available at http://apache.org/foundation/board/calendar.html

ASF Infrastructure –our distributed team on four continents keeps the ASF's infrastructure running around the clock.
 - 6M weekly checks yielded a sunny 99.73% uptime over the past week http://status.apache.org/ Check against established SLAs http://status.apache.org/sla/

Apache AsterixDB (incubating) –a Big Data Management System (BDMS) that can store, index, manage, and query semi-structured data at scale on a cluster.
 - Apache AsterixDB 0.8.7-incubating released https://asterixdb.incubator.apache.org/download.html

Apache PDFBox™ –an Open Source Java tool for working with PDF documents.
 - Apache PDFBox 2.0.0 RC1 released http://pdfbox.apache.org/download.cgi

Apache Sling™ –a web framework that uses a Java Content Repository, such as Apache Jackrabbit, to store and manage content.
 - Apache Sling 8 released https://sling.apache.org/downloads.cgi

Apache Stratos™ –a PaaS framework that is designed to address the various challenges currently faced in the Cloud PaaS space.
 - Apache Stratos 4.1.4 released http://www.apache.org/dyn/closer.cgi/stratos

Apache Tomcat™ –an Open Source software implementation of the Java Servlet, JavaServer Pages, Java Unified Expression Language and Java WebSocket technologies.
 - Apache Tomcat 7.0.65 released http://tomcat.apache.org/download-70.cgi


Did You Know?

 - Did you know that 16 Apache projects won Bossie awards? Application Development Tools --Cassandra and Cordova; Big Data -- Drill, Flink, HBase, Hive, Kafka, Mesos, NiFi, Spark, and Storm, as well as Apex, Kylin, Ranger, and Zeppelin (all incubating); Data Center and Cloud Software --Mesos http://www.infoworld.com/article/2984437/open-source-tools/bossies-2015-the-best-of-open-source-software-awards.html

 - Did you know that the Law Library of Congress, the National Hockey League, and Unilever use Apache Cayenne? http://cayenne.apache.org/

 - Did you know that Bloomberg uses Apache HBase, Mesos, and Spark to serve hundreds of thousands of sophisticated financial users? https://blogs.apache.org/hbase/date/20150903
