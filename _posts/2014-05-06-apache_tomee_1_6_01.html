Apache TomEE is happy to announce the security related release of Apache TomEE 1.6.0.2 which is still based on Apache Tomcat 7.0.53 - The principal focus of this release is to provide an update to CXF version 2.6.14, which fixes several security issues.

Complete details of the issues that have been fixed can be found at the following link:

http://cxf.apache.org/security-advisories.html

The Apache TomEE Release 1.6.0.2 files can be found here:

http://tomee.apache.org/downloads.html

A complete Changelog can be viewed here:

http://tomee.apache.org/tomee-1.6.0.1-release-notes.html
