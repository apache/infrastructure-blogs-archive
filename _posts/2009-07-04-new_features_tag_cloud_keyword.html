I added a "tagcloud" module to the contributions area. It is visible in the defaultfiredocs publication. You can now configure the meta data to be indexed as keywords (complete phrases incl. spaces as index terms). This is especially useful for using phrases as tags. Check out the meta data docs for the 2.0.x branch in the Zones docu publication.

Have fun!
