The Apache OpenWebBeans Team is proud to announce the release of Apache OpenWebBeans-1.6.2

Apache
OpenWebBeans is an Apache License v2 licensed implementation of the
JSR-346 CDI-1.2 specification.

OpenWebBeans has a modular structure and
provides Dependency Injection scaling from Java SE environments up to
Java EE7 server clusters with complicated ClassLoader hierarchies or OSGi
environments.

OpenWebBeans-1.6.2 implements the CDI-1.2 API, passes the JSR-330 TCK and the JSR-346 standalone and EE7 webprofile TCKs.

Distribution packages can be downloaded from

http://www.apache.org/dyn/closer.cgi/openwebbeans/1.6.2/

The release is also available in maven.central

http://repo1.maven.apache.org/maven2/org/apache/openwebbeans/

More info can be found at

http://openwebbeans.apache.org

The Apache OpenWebBeans Team
