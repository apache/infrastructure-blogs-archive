Version 3.3.0 includes new features, bug fixes and security fixes.



Enhancements around:

  • Room is displayed without overlap in IE
  • Multiple display issues
  • Wb room element can now be hidden



Security fixes:

  • Chat
  • All requests via security headers
  • More secure password processing rules and storage
  • More strict rules for uploaded files
  • SQL injection in web services



CVE reports have been submitted and addressed, complete list see changelog.

Thanks to the CVE reporters!



Download and documentation available at: openmeetings.apache.org/downloads.html
