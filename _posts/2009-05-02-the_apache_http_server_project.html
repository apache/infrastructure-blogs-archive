Not only does the Apache HTTP Server power more
than 106 million Websites worldwide, an astounding 89% of Linux
Journal readers have chosen the “competitor-less” Apache HTTP Server as their
favorite Web server, according to the 2009
Reader’s Choice Awards.

We were tickled to read, “As with last year, the thought
arises in Favorite Web Server category of whether we should just ask ‘Do you
use the Apache Web server, yes or no?’ and leave it at that.” According to
Associate Publisher Mark Irgang, “Apache is always in there.”

Congratulations to the Apache
HTTP Server Project for more than 16 years of technical
excllence!
