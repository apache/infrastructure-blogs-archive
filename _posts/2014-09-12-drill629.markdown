  <p class="MsoNormal" style="line-height: 15pt; background-image: initial; background-attachment: initial; background-size: initial; background-origin: initial; background-clip: initial; background-position: initial; background-repeat: initial;"><span style="font-size: 10.5pt; font-family: Arial; color: #333333;">It is our pleasure to announce the 0.5.0 release of Apache
Drill.&nbsp; This is Drill’s first beta
release and the second in our iterative monthly release cycle. It includes more than <a href="https://issues.apache.org/jira/secure/ReleaseNote.jspa?projectId=12313820&amp;version=12324880">100 issues</a> addressed
since last month’s release and more than 1,000 addressed since Drill’s
inception, this is a great release to start exploring your data, wherever and
whatever it is. <o:p /></span></p> 
  <p class="MsoNormal" style="line-height: 15pt;"><span style="font-size: 10.5pt; font-family: Arial; color: #333333;">For more background on what Drill is about, check out the <a href="http://incubator.apache.org/drill/overview.html">Drill overview</a> or <a href="https://cwiki.apache.org/confluence/display/DRILL/Apache+Drill+in+10+Minutes">Drill
in 10 minutes</a>. The 0.5.0 release
builds upon the huge 0.4.0 release so you should refer to <a href="https://blogs.apache.org/drill/entry/announcing_apache_drill_0_4">last
month’s release</a> for information on all the functionality available. Notable features included in 0.5.0 include
the following:</span><span style="font-size: 10.5pt; font-family: Arial; color: #333333;"><o:p /></span></p> 
  <p class="MsoNormal" style="margin-top: 7.5pt; line-height: 15pt; background-image: initial; background-attachment: initial; background-size: initial; background-origin: initial; background-clip: initial; background-position: initial; background-repeat: initial;"> </p> 
  <ul> 
    <li><span style="color: #333333; font-family: Arial; font-size: 10.5pt; line-height: 15pt;">Drill now uses the Hadoop 2.4.1 APIs.&nbsp; </span><span style="color: #333333; font-family: Arial; font-size: 10.5pt; line-height: 15pt;">This includes upgrading Parquet to use direct
memory and the ability to write larger Parquet files when using CREATE TABLE AS.</span></li> 
    <li><span style="color: #333333; font-family: Arial; font-size: 10.5pt; line-height: 15pt;">Improved JOIN planning when using HBase tables based on row
count approximations using region level statistics.</span></li> 
    <li><span style="color: #333333; font-family: Arial; font-size: 10.5pt; line-height: 15pt;">Improved handling of large sorts and out of memory conditions.</span></li> 
    <li><span style="color: #333333; font-family: Arial; font-size: 10.5pt; line-height: 15pt;">JSON projection pushdown, an all text JSON mode and boolean
short circuit.</span><span style="color: #333333; font-family: Arial; font-size: 10.5pt; line-height: 15pt;"> </span><span style="color: #333333; font-family: Arial; font-size: 10.5pt; line-height: 15pt;">Each of these features
allow more flexibility when interacting with complicated JSON files</span></li> 
    <li><span style="color: #333333; font-family: Arial; font-size: 10.5pt; line-height: 15pt;">Substantial improvements in SELECT * handling when interacting
with schemaless data sources.</span></li> 
    <li><span style="color: #333333; font-family: Arial; font-size: 10.5pt; line-height: 15pt;">Creation of a self contained JDBC JAR file to ease access to
Drill from JDBC tools.</span></li> 
    <li><span style="color: #333333; font-family: Arial; font-size: 10.5pt; line-height: 15pt;">Fully distributed execution of all basic aggregates including
standard deviation and avg.</span></li> 
  </ul> 
  <p> </p> 
  <p class="MsoNormal" style="line-height: 15pt; background-image: initial; background-attachment: initial; background-size: initial; background-origin: initial; background-clip: initial; background-position: initial; background-repeat: initial;"><span style="font-size: 10.5pt; font-family: Arial; color: #333333;">Drill will continue on its march towards GA with upcoming
monthly releases continuing to harden and expand Drill’s capabilities and
performance. Check out the <a href="https://cwiki.apache.org/confluence/display/DRILL/Apache+Drill+0.5.0+Release+Notes">release
notes</a>, <a href="http://incubator.apache.org/drill/download.html">download</a>
it, or better yet, <a href="https://github.com/apache/incubator-drill/tree/0.5.0-incubating">make
your own fork</a> and contribute back to the community. Together, we can make data available to everyone, anywhere.</span></p> 
  <p>-The Apache Drill Team&nbsp;&nbsp;</p>
