The Isis team is pleased to announce the release of Apache Isis Core 1.1.0, Isis Shiro Security 1.1.0, Isis Wicket Viewer 1.1.0 and Quickstart (Wicket/Restful/JDO) Archetype 1.0.2 released.
New and notable features are:
  • upload/download of files in the Wicket viewer
  • support for bulk actions in the Wicket viewer
  • improved extensibility of Wicket viewer (eg for gmap3 support)
  • support for LDAP authorization through a Shiro Realm implementation
Full release notes are available at [1], [2], [3], [4] on the Isis website.
You can access this release directly from the Maven central repo [5], 
or download the release and build it from source [6].
Enjoy!
-The Isis team
